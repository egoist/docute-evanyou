Add the exact same effect in evanyou.me to your docute landing page.

## Usage

Add `docute-evanyou` to `index.html` in your docute docs:

```js
<script src="https://unpkg.com/docute-evanyou"></script>
```

And enable it in `config.js`:

```js
self.$config = {
  plugins: [
    evanyou()
  ]
}
```
